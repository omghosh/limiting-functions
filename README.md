This repository contains the data and code for the manuscript "Genotype-fitness mapping of adaptive mutants reveals shifting low-dimensional structure across divergent environments," by Ghosh, Kinsler, Good, and Petrov. 

The data is stored in the data directory, and contains barcode counts and the inferred fitness values, which were obtained using code in mle_fitness_inference. 

To generate figure subpanels, run the python files labeled figX_revised.py, where X is the figure subpanel of interest (ie X = 2a).
